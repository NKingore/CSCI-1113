# mynotes
Use 1st time to get copy of repository (git@github.com:NKingore/CSCI-1113.git):
``` shell
git clone git@github.com:NKingore/CSCI-1113.git
```
Add new code/changes
```shell
# show status of local repository
1) git status

# add files I want to commit
2) git add filename

# commit the files and add message with changes 
3) git commit -m "message here"

# push commited files to remote origin (github website)
4) git push -u origin main
```
